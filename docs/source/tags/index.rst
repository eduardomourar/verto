Available Tags
#######################################

This document covers how to use the available tags within Markdown text, and also the conditions for custom HTML for each tag.

Using Tags
=======================================

- Tags should always be separated with newlines before and after each tag.
- If a tag requires both a start and end tag (for example: panels) and one tag is missing then a ``TagNotMatchedError`` Exception will be thrown.
